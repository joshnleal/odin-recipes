# odin-recipes

This project is a basic recipe website.

The website will contain a main index page that will link to a few recipes.

At the end of this project, I will have learned how to work with Git and HTML.
Using Git in this project will teach me the importance of Version Control Systems, file structures, and writing 
clear and concise commit messages that can be used when debugging and maintaining the website.
With HTML, I will have learned how to write HTML semantically, using tags such a anchors, images, ordered lists, etc. and attributes like
href, src, alt, rel, etc.
